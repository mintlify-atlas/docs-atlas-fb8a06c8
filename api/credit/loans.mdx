## Create Loan

<CodeGroup>
```typescript POST /loan
import { usePostLoan } from '@/services/hooks/mutations/use-post-loan';

const { mutate: createLoan } = usePostLoan();

createLoan({
  userId: '79f970d7-3b2f-492c-a358-6c1f1c2fd429',
  ratio: '1.5',
  size: '1',
  typeId: 'ETH_USDT'
});
```
</CodeGroup>

Create a new collateralized loan. The loan is immediately active upon creation and begins accruing interest based on the collateral ratio tier.

### Request Body

<ParamField body="ratio" type="string" required>
  Target collateral ratio for the loan. Must meet the minimum collateral ratio requirements for the loan type. Higher ratios result in lower APR rates based on tier structure.
  
  Example: `"1.5"` for 150% collateralization
</ParamField>

<ParamField body="size" type="string" required>
  The loan amount in the size currency (e.g., USDT). Must be within the loan type's min/max size limits.
  
  Example: `"1"` for 1 USDT
</ParamField>

<ParamField body="typeId" type="string" required>
  The loan type identifier specifying the collateral and size currencies.
  
  Available types: `ETH_USDT`, `BTC_USDT`
</ParamField>

### Response

<ResponseField name="typeId" type="string">
  The loan type identifier (e.g., `ETH_USDT`)
</ResponseField>

<ResponseField name="operation" type="object">
  <ResponseField name="id" type="string">

  </ResponseField>
  
  <ResponseField name="creatorId" type="string">

  </ResponseField>
</ResponseField>

<ResponseField name="sizeCurrencyId" type="string">
  Currency of the loan amount (e.g., `USDT`)
</ResponseField>

<ResponseField name="collatCurrencyId" type="string">
  Currency of the collateral (e.g., `ETH`, `BTC`)
</ResponseField>

<ResponseField name="status" type="string">

</ResponseField>

<ResponseField name="size" type="string">

</ResponseField>

<ResponseField name="collat" type="string">
  Current collateral amount in collateral currency

<ResponseField name="repayed" type="string">
  Total amount repaid so far
</ResponseField>

</ResponseField>

<ResponseField name="interest" type="string">

## Repay Loan

<CodeGroup>
```typescript POST /loan/{opId}/repay
import { usePostLoanManager } from '@/services/hooks/mutations/use-post-loan-manager';

const { mutate: repayLoan } = usePostLoanManager('repay');

repayLoan({
  userId: '79f970d7-3b2f-492c-a358-6c1f1c2fd429',
  opId: '2c7ce71e-8dc8-4dd2-97e1-3d9716a1874d',
  amount: '1.992138092931196373'
});
```
</CodeGroup>

Repay part or all of an active loan. When the debt is fully repaid, the loan status changes to `REPAYED` and remaining collateral is returned to the user.

### Path Parameters

<ParamField path="opId" type="string" required>
  The loan operation ID to repay
</ParamField>

### Request Body

<ParamField body="amount" type="string" required>
  The amount to repay in the size currency. Can be partial or full repayment.
  
  Example: `"1.992138092931196373"` to repay 1.99 USDT
</ParamField>

### Response

Returns the updated loan object with the same structure as the Create Loan response. The `repayed` field will be updated, and `debt` will be reduced accordingly.

## Liquidate Loan

<CodeGroup>
```typescript POST /loan/{opId}/liquidate
import { usePostLoanManager } from '@/services/hooks/mutations/use-post-loan-manager';

const { mutate: liquidateLoan } = usePostLoanManager('liquidate');

liquidateLoan({
  userId: '79f970d7-3b2f-492c-a358-6c1f1c2fd429',
  opId: 'edaa7bd5-00d6-40b6-8b27-f21e3b06c561',
  amount: '10.085159966953139616'
});
```
</CodeGroup>

Liquidate collateral to repay part of the loan. This sells collateral at current market price to reduce debt, useful when you want to maintain your loan position while reducing risk.

### Path Parameters

<ParamField path="opId" type="string" required>
  The loan operation ID to liquidate
</ParamField>

### Request Body

<ParamField body="amount" type="string" required>
  The amount of debt to repay through liquidation (in size currency). The system will liquidate enough collateral at current market price to cover this amount plus liquidation fees.
  
  Example: `"10.085159966953139616"` to liquidate collateral worth ~10 USDT
</ParamField>

### Response

Returns the updated loan object. The `lastUpdate.type` will be `REPAY_COLLAT` and will include:
- `liqCollat`: Amount of collateral liquidated
- `liqCollatValue`: Value of liquidated collateral in size currency
- `price`: Execution price
- `fee`: Liquidation fee charged

## Get Loan History

<CodeGroup>
```typescript GET /loan/{opId}/history
import { useLoanHistoryEvents } from '@/services/hooks/use-loan-history-events';

const { data: historyEvents } = useLoanHistoryEvents({
  userId: '79f970d7-3b2f-492c-a358-6c1f1c2fd429',
  opId: '389ae85b-7968-4086-b7e3-2b53dc4fe6ed',
  enabled: true
});
```
</CodeGroup>

Retrieve the complete history of events for a loan, including collateral changes, repayments, APR updates, and liquidations.

### Path Parameters

<ParamField path="opId" type="string" required>
  The loan operation ID to retrieve history for
</ParamField>

### Response

<ResponseField name="events" type="array">
  Array of loan history events in chronological order
  
  <ResponseField name="createdAt" type="string">
    Timestamp when the event occurred (ISO 8601)
  </ResponseField>
  
  <ResponseField name="type" type="string">
    Event type:
    - `ADD_COLLAT`: Collateral was added
    - `REM_COLLAT`: Collateral was removed
    - `REPAY_SIZE`: Loan repayment in size currency
    - `REPAY_COLLAT`: Debt repaid via collateral liquidation
    - `UPDATE_APR`: APR tier changed due to ratio change
  </ResponseField>
  
  <ResponseField name="data" type="object">
    Event-specific data fields
    
    <ResponseField name="currencyId" type="string" optional>
      Currency involved in the event (for ADD_COLLAT, REM_COLLAT, REPAY_SIZE)
    </ResponseField>
    
    <ResponseField name="amount" type="string" optional>
      Amount of currency involved
    </ResponseField>
    
    <ResponseField name="liqCollat" type="string" optional>
      Amount of collateral liquidated (for REPAY_COLLAT)
    </ResponseField>
    
    <ResponseField name="liqCollatValue" type="string" optional>
      Value of liquidated collateral in size currency
    </ResponseField>
    
    <ResponseField name="repayed" type="string" optional>
      Amount repaid from liquidation
    </ResponseField>
    
    <ResponseField name="price" type="string" optional>
      Execution price for liquidation
    </ResponseField>
    
    <ResponseField name="fee" type="string" optional>
      Fee charged for the liquidation
    </ResponseField>
    
    <ResponseField name="apr" type="string" optional>
      New APR value (for UPDATE_APR events)
    </ResponseField>
  </ResponseField>
</ResponseField>

  ratio: string;

  apr: string;
  tierNum: number;

  initialAPR: string;
